비용 관리·근태·정산 기능 업데이트

업데이트 | 2025년 2월 25일

insa-IT는 보다 정확하고 효율적인 인사 운영을 위해 비용 관리, 근태 설정, 정산 기능을 개선하였습니다. 이번 업데이트는 관리자와 실무자가 체감할 수 있는 편의성 중심으로 구성되었습니다.

1. 비용 사용 내역 승인 처리 단축

비용 승인자는 더 빠르게 승인할 수 있도록 UI 개선과 함께 ‘승인 대기’만 필터링하여 처리할 수 있는 기능이 추가되었습니다.

[ 이미지: 비용 승인 필터링 기능 화면 ]

2. 유연근무제 근무시간 자동 계산

근태 설정에서 ‘시간 단위 유연근무제’를 사용하는 경우, 업무 시작/종료 시간만 설정하면 자동으로 실근무 시간이 계산되어 별도 입력이 필요 없습니다.

[ 이미지: 유연근무제 근태 설정 화면 ]

3. 정산서 내역 일괄 다운로드 기능

인사담당자가 월 단위 정산자료를 일괄로 다운로드할 수 있도록 [정산서 내보내기] 기능이 신설되었습니다. PDF 또는 엑셀로 저장 가능하며, 양식도 선택할 수 있어 보고용 자료 정리가 편리해졌습니다.

[ 이미지: 정산서 다운로드 화면 예시 ]

📌 관리자 팁:
이번 업데이트는 관리자 및 회계 담당자의 정기 업무 시간을 절감하고, 승인 처리 정확도를 높이기 위한 목적이 큽니다.
기능별 도움말은 insa-IT 도움센터에서 확인하실 수 있습니다.
