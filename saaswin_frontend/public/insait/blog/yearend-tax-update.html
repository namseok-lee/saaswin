2024년 귀속 연말정산 업데이트

업데이트 소식 | 2025년 1월 16일

2024년 귀속 연말정산을 더욱 정확하고 빠르게 처리할 수 있도록, insa-IT가 연말정산 프로세스를 개선했습니다. 사용자 편의성과 절세 가이드를 강화한 것이 주요 포인트입니다.

1. 제출 항목 미리보기 및 체크 기능

연말정산 제출 항목을 미리 확인하고, 누락 여부를 스스로 점검할 수 있도록 체크리스트 기능이 추가되었습니다. 실수 없이 서류 제출이 가능해졌습니다.

[ 이미지: 제출 항목 체크 기능 화면 ]

2. 절세포인트 요약 제공

절세에 유용한 항목을 요약해 알려주는 ‘절세포인트 안내’ 기능이 신설되어, 구성원 스스로 절세 전략을 세우는 데 도움을 줍니다.

[ 이미지: 절세포인트 안내 화면 ]

3. 관리자용 검토 현황 모니터링

관리자 화면에서는 구성원별 제출 현황, 검토 여부, 이상 항목 유무 등을 직관적으로 확인할 수 있어 정산 업무가 더욱 수월해졌습니다.

[ 이미지: 관리자 검토 화면 예시 ]

📌 이번 연말정산은 제출자의 자가 점검 → 관리자 검토 → 일괄 정산까지 한 흐름으로 이어지도록 설계되었습니다.
구성원과 관리자 모두에게 효율적인 프로세스를 제공합니다.
