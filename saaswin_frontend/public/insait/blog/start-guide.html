insa-IT 시작 가이드 [시스템총괄모드]

가이드 | 2025년 4월 10일

insa-IT의 시스템총괄모드는 조직 전체의 구성과 운영 체계를 처음부터 설계할 수 있는 관리자 전용 설정 모드입니다. 인사담당자, 운영관리자 등 초기 설정을 총괄하는 분들을 위해 구성된 가이드입니다.

1. 시스템총괄모드 진입 방법

insa-IT에 로그인 후, 상단 메뉴에서 [관리자 전용 메뉴] → [시스템총괄모드]를 클릭합니다. 해당 모드에서는 기본 정보 설정부터 역할, 부서, 권한 체계 구성까지 통합적으로 관리할 수 있습니다.

[ 이미지: 시스템총괄모드 진입 위치 안내 ]

2. 조직도 구조 구성

조직도는 팀, 파트, 사업부 등 실제 운영 단위에 맞게 자유롭게 구성할 수 있습니다. 각 조직별로 소속 인원, 관리자, 하위조직을 지정해 체계적인 조직 운영 기반을 마련하세요.

[ 이미지: 조직도 설정 화면 예시 ]

3. 사용자 계정 및 역할 설정

구성원별 계정 등록 후, 역할(Role)과 권한(Permission)을 세부적으로 설정할 수 있습니다. ex) HR 관리자, 회계 담당자, 일반 사용자 등 역할별 접근 권한을 다르게 구성 가능

[ 이미지: 사용자 권한 설정 화면 ]

4. 초기 필수 설정 점검사항

  • ✔ 사업장 기본 정보 입력
  • ✔ 부서 및 사용자 그룹 구성
  • ✔ 승인 라인 기본값 지정
  • ✔ 출퇴근 규칙 설정
  • ✔ 메시지/알림 설정 여부 확인

설정이 완료되면, 각 팀에서는 insa-IT의 다양한 기능을 실무에서 바로 사용할 수 있습니다.
시스템총괄모드는 초기 셋업을 위한 핵심 기능이므로, 해당 가이드를 참고하여 꼼꼼히 구성해 주세요.
